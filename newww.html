Luna Assistant

Welcome to Luna Assistant, your virtual assistant powered by Python and Eel.

Features

  • Voice recognition and commands
  • Task automation (open applications, take screenshots, etc.)
  • Email sending and file sharing capabilities
  • Integration with external services like Wikipedia, news, and more
  • Interactive games and entertainment

About the Project

Luna Assistant is a personal assistant project developed using Python, Eel, and various external libraries. It aims to provide a seamless and interactive experience for users to perform tasks using voice commands.

Getting Started

Click the "Start Luna Assistant" button below to initiate the assistant. You can then interact with Luna using voice commands and explore its features.
